# 032: 日本語コーパスでの学習デモ

## コーパス

芥川龍之介「蜘蛛の糸」（青空文庫）
- 約2,900文字
- ユニーク文字: 311種（vocab_size=314, PAD+UNK含む）

## モデル設定

| パラメータ | 値 |
|---|---|
| d_model | 64 |
| n_heads | 4 |
| d_ff | 128 |
| n_layers | 2 |
| dropout | 0.1 |
| seq_len | 64 |
| epochs | 50 |
| lr | 0.001 |
| 勾配クリッピング | max_norm=1.0 |
| LR スケジュール | 10% warmup + cosine decay |

## 結果

### 生成テキスト

**プロンプト「極楽」:**
> 極楽の蓮の山当って、こでございます。御釈迦男がございまりますと、この様はなりもそうますとす。

**プロンプト「蜘蛛の糸」:**

## 学習の改善効果

Step 29-31 で追加した3つの機能の効果：

### 勾配クリッピング (Step 29)
- 勾配爆発を防止
- NaN 発散なしで安定して学習完了

### LR スケジュール (Step 30)
- Warmup で初期の不安定を回避
- Cosine decay で後半の振動を抑制

### Dropout (Step 31)
- 生成テキストの崩壊（同じ文字の無限繰り返し）が抑制
- 以前: 「足足足足足足足」のような崩壊
- 現在: 崩壊せず文の体裁を保つ

## 性能

- `cargo run --release` で **数秒** で学習完了
- release ビルドによる最適化が支配的

## 残る課題

- 文法的に正しい文を安定して生成するには、より大きなモデルとより多くのデータが必要
- バッチ処理がないため、勾配のノイズが大きい
- 文字レベルのため、長い依存関係の学習が難しい

## まとめ

フレームワークなし、`rand` のみの依存で、日本語テキストの学習と生成ができるようになった。小さなモデルでも原文の語彙・文体のパターンを捉えている。
